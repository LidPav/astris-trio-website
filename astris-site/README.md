# Astris Trio – Website (11ty + Decap CMS)

## Lokal starten
1. Node.js installieren (LTS).
2. In diesem Ordner:

npm install
npm run start

3. Im Browser: http://localhost:8080

## Deploy (Netlify)
1. Repo auf GitHub anlegen und pushen.
2. Netlify: New site from Git → Repo wählen.
3. Build command: npm run build
4. Publish directory: _site

## Editor (Decap CMS)
- Nach dem Deploy ist der Editor unter /admin/ erreichbar.
- In Netlify: Identity aktivieren und Git Gateway aktivieren.
- Danach können Nicht-Techniker:innen Konzerte, Presse sowie Stammdaten (Musiker:innen/Repertoire) über Formulare pflegen.
